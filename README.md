# Проект Gulp для автоматизації розробки

Цей проект створено для домашнього завдання з курсу Hillel Fullstack JS.

## Можливості

Даний проект включає налаштування Gulp для автоматизації наступних процесів:

- Компіляція SCSS в CSS
- Оновлення сторінки в реальному часі за допомогою Browser-Sync
- Автоматичне додавання вендорних префіксів
- Форматування SCSS коду за допомогою CSScomb
- Мінімізація та оптимізація CSS
- Управління медіа-запитами

## Структура проекту

```
.
├── assets/
│   ├── css/          # Скомпільовані CSS файли
│   ├── js/           # JavaScript файли
│   └── scss/         # SCSS файли
├── .config.json      # Конфігурація для CSScomb
├── .gitignore        # Git ігнорування файлів
├── gulpfile.js       # Налаштування Gulp
├── index.html        # Головна сторінка
└── package.json      # Залежності проекту
```

## Встановлення та використання

1. Клонуйте репозиторій
2. Встановіть залежності:

```
npm install
```

3. Доступні команди:

```
npm start     # Запуск режиму спостереження
npm run build # Збірка проекту
```

## Технології

- Gulp 4
- SCSS
- Browser-Sync
- Autoprefixer
- CSSnano
- CSScomb
